Identidad de Marca

Appgile se especializa en desarrollos web de plugins para corporaciones que incluyen los plugins en sus páginas asociadas. Por ejemplo, plugin de calculadora de financiamiento para páginas web de vehículos.

Misión

Desarrollar soluciones innovadoras y plugins especializados que potencien las capacidades digitales de las corporaciones, facilitando la integración de herramientas avanzadas en sus plataformas web.

Espacio de Seguridad del Logo

El logo debe tener un espacio de seguridad equivalente a la altura de la letra "A" del símbolo en todos sus lados. Esto asegura que el logo maximice su visibilidad e impacto.

Directrices de Uso

✓ Uso Correcto

  • ✓ Usar el logo completo cuando sea posible
  • ✓ Mantener el espacio de seguridad
  • ✓ Usar las versiones en blanco sobre fondos oscuros
  • ✓ Respetar las proporciones originales
  • ✓ Usar los colores oficiales de la marca

✗ Uso Incorrecto

  • ✗ No distorsionar o estirar el logo
  • ✗ No rotar el logo
  • ✗ No usar colores incorrectos
  • ✗ No aplicar sombras o efectos
  • ✗ No desenfocar el logo
